Introdução


Conceitos básicos

O suporte a linguagens de programação do Notes é totalmente extensível e exterior ao Notes. Através dele você pode customizar o suporte de qualquer linguagem (por exemplo, modificando a forma como a coloração é feita) ou até mesmo adicionar linguagems que não vem por padrão com o Notes.

No Notes as linguagens de programação são chamadas de "Tipos de Arquivo" (em inglês, File Types). Este é um termo melhor, já que nada impede que o Notes faça coloração de certos arquivos de configuração (arquivos INI, por exemplo) e outros formatos de arquivo. Cada Tipo de Arquivo possui uma pasta só sua em "..\Pasta Do Profile Atual\FileTypes\Nome Do Tipo De Arquivo\". A pasta do profile atual fica em "C:\Documents And Settings\Nome do Usuário\ Dados de Aplicativos\Notes2004\Profiles\Nome Do Profile" no Windows 2000/XP e em "C:\Windows\Application Data\Notes2004\Profiles\Nome Do Profile" no Windows 98/Me. Por exemplo, a pasta com suporte a PHP poderia ficar em: 
C:\Documents and Settings\joao\Dados de aplicativos\Notes2004\profiles\default\filetypes\PHP

Arquivos e Pastas

A pasta que contém o suporte a uma linguagem de programação tem a seguinte estrutura: 
  ..\Library\ [1]
  ..\Templates\ [2]
  ..\ext.dat [3]
  ..\Nome do Tipo de Arquivo.nhl [4]
  ..\Nome do Tipo de Arquivo.nhs [5]
  ..\Nome do Tipo de Arquivo.xml [6]
  ..\Nome do Tipo de Arquivo.nsi [7]

O item [1], chamado "Library", é a pasta onde ficam os items da biblitoeca para aquele tipo de arquivo. Você pode criar os items usando o painel biblioteca do Notes, então não se preocupe em editar nada à mão. O item [2] é a pasta onde ficam guardados os templates para novos arquivos (você pode editar eles usando os diálogos do Notes). O item [3], "ext.dat", é um arquivo onde ficam as extensões para o tipo de arquivo. Estes precisam ser editados pelo diálogo de preferências do Notes, pois o diálogo "compila" o conteúdo deste arquivo em outros arquivos. O item [4] e o item [5] são os arquivos que fazem a coloração do código. O item [6] é o arquivo onde ficam guardadas as opções para o tipo de arquivo. Ele pode ser criado e editado pelo diálogo de opções do Notes, porém algumas opções avançadas só podem ser editadas a mão. O item [7] é o arquivo que contém as regras da engine de SmartIndent e só pode ser editado manualmente.
