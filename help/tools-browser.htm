Fazendo preview de sites com o Notes


Sites em HTML

Para fazer o preview de um site em HTML no navegador, vá no menu executar >> configurar ferramentas e escolha a opção "novo item". Dê um nome para o item. Logo depois, clique no botão com o símbolo de "mais" (+) ao lado direito do campo chamado "executar" e escolha a opção "procurar arquivo". Procure o executável do seu navegador (exemplo : C:\Arquivos de programas\mozilla.org\Mozilla\mozilla.exe) e clique em "abrir". Certifique-se que no campo "argumentos" esteja escrito "%file%" com as aspas. Dê ok para salvar as suas configurações. Pronto, basta usar o novo item que aparecerá no menu para fazer o preview do site site HTML no navegador. Você pode usar os mesmos passos para adicionar outros compiladores. Você pode também associar uma tecla de atalho que ao ser acionada chamará o navegador - basta voltar ao diálogo para configurar ferramentas, selecionar o item e clicar no botão "tecla de atalho".

Sites dinãmicos feitos em PHP, ASP, JSP, Perl, etc.

Sites dinâmcios precisam ser executados através de um servidor. Você pode usar os mesmos passos descritos acima para as páginas normais, porém você precisa configurar o campo "argumentos" de uma forma um pouco diferente, preenchendo ele assim: 
http://meu_servidor/minha_pagina/%filename%
Onde:
meu_servidor: é o seu servidor de teste
minha_pagina: é o nome da pasta raiz do seu projeto
%filename%: é uma macro que será substituída pelo nome do arquivo (Ex: index.php)

Um exemplo de argumento seria: 
http://localhost/sitedonotes/%filename%
