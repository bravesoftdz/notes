Conceitos básicos


A interface

O Notes é composto por uma interface bastante semelhante a de outros programas. Os elementos da interface que você já deve conhecer bem são o menu, a barra de ferramentas e a barra de status. Além deles, o Notes tem uma série de tabs e painéis.

Tabs (também chamadas de "abas") contém normalmente um editor onde você pode editar um arquivo. As tabs podem também conter a StartPage e outros elementos. Esta disposição em tabs permite que você edite centenas de arquivos em apenas uma janela do Notes.

Painéis são pequenas janelas que aparecem a esquerda, direita ou em baixo da área das tabs. Você pode mostrar/esconder os painéis clicando no pequeno botão com uma flecha no centro que aparece na linha que separa os painéis do editor. Os painéis também são dispostos em tabs (abas). Você pode mudar um painel de lugar clicando com o botão direito do mouse na tab do painel e escolhendo no menu de contexto a nova localização.

Tipos de arquivos

Como já foi dito, o Notes não conhece nenhuma linguagem por padrão. Tipos de arquivo são um conjunto de arquivos que fazem o Notes reconhecer uma linguagem, colorir a sua sintaxe, oferecer uma biblioteca de códigos para ele, identar o seu código, etc. Exemplo de tipos de aqruivo são ASP, HTML, Delphi, C-C++, CSS, etc.

Profiles

O Notes guarda todas as suas configurações dentro de um profile. Um profile é como um grupo de configurações que são mantidas juntas. Você pode ter vários profiles: um para criar webpages, outro para programa em Java, outro para escrever XML, etc. Claro, você pode fazer tudo isto com um profile só, mas usando profile você pode escolher configurações que funcionem melhor para certos tipos de trabalhos. O Notes precisa de ao menos um profile para funcionar, já que é o profile que comanda o funcionamento do Notes. Durante a instalação o Notes cria um profile padrão. Você pode gerenciar os profiles em Opções >> Configurar Profiles.
