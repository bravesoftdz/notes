O que é e o que não é o Notes


O Notes é um editor para programadores. O Notes não é um visualizador de logs, um processador de textos, um editor para arquivos gigantes ou um editor para qualquer coisa para além de código fonte ou webpages.

O Notes é um editor para todas as linguagems de programação e para nenhuma em especial. Isto quer dizer que ele não tem e não terá features específicas para a linguagem X ou Y, como muitos outros editores que se dizem genéricos possuem. Isto quer dizer que ele não trata a linguagem C ou Pascal ou XML ou qualquer outra de forma especial - ele não conhece nenhuma linguagem por padrão, não há nenhum tipo de arquivo hardcoded.

O Notes é um editor simples de usar. Isto quer dizer que ele não quer ser tudo para todo mundo nem a solução de todos os problemas. Isto quer dizer que ele não tem e não terá opções e features que só 0,5% dos usuários vão usar. Existem outros editores por aí que tem todas as opções de todos os editores que já foram criados - o Notes não é mais um destes.

O Notes é software livre e é desenvolvido por um grupo de voluntários que formam a Equipe do Notes. Ninguém ganha nenhum tipo de remuneração por ajudar o Notes. O Notes e sua equipe não contratam ninguém nem possuem dinheiro para coisa alguma (por favor, não mande seu currículo para nós pedindo emprego como alguns já o fizeram, nós não temos como pagar nada nem ninguém). Não é financiado por qualquer fundação, instituição, empresa, companhia multinacional, etc. como ocorre com certos projetos. Isto quer dizer que se você que usa e gosta do Notes não contribuir com o Notes, ele não ficará nem um pouco melhor. O Notes depende da sua ajuda para crescer!.
