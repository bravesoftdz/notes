Desenvolvendo Softwares


O Notes pode ser usado e é usado por muitos programadores para desenvolver software. Apesar de não vir com compiladores, é fácil integrar o Notes ao(s) seu(s) compilador(es) preferido(s) e a outras ferramentas de desenvolvimento. Veja como configurar o seu compilador no Notes.

Além da integração com compiladores, o Notes oferece um editor com coloração de código e várias features para programar mais rápido, como o SmartIndent. Leia o caítulo sobre edição de arquivos para saber mais sobre estas features.

Outras features que serão muito úteis são o suporte a macros escritas em JavaScript e a configuração das teclas de atalho. Você pode usar o suporte a macros para automatizar certas partes do desenvolvimento (como criar uma classe, achar a definição de uma função, completar código, etc.). O Notes possuí também uma biblioteca de códigos para cada linguagem (acessível pelo painel biblioteca) que você pode usar para guardar aqueles trechos de código que você vive digitando.
