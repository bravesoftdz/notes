Compilando com o Notes


O Notes não tem um compilador padrão nem vem com nenhum compilador, mas você pode integrar virtualmente qualquer compilador ao Notes usando o menu executar. Se você não sabe que compilador usar ainda, dê uma olhada neste site para uma lista de compiladores freeware.

O menu executar

O menu executar é um menu altamente configurável. Cada item do menu serve para executar o arquivo que você está editando em um compilador ou em algum outro tipo de programa. Digamos que você queira compilar um arquivo de código fonte Java para bytecode (o processo é o mesmo para quase todos os compiladores). Vá em Executar > Configurar menu. No diálogo que aparece, clique em novo. Chame o novo item de "Compilar Java para bytecode". No campo executar, digite o caminho para o compilador (javac.exe no caso do java). Você não precisa digitar o caminho à mão. Clique no botão que possuí um sinal de mais (+) ao lado do campo. Vai aparecer um menu. Nele escolha "Procurar arquivo" e procure pelo executável do compilador. No campo argumentos, normalmente basta iserir a variável %file%. Os items do menu executar suportam muitas variáveis que são substituída quando o item vai ser usado por as informações verdadeiras. %file% é substitído pelo arquivo que você estiver editando.


Erros de compilação

O Notes é capaz de capturar a saída dos compiladores e mostrar ela no painel saída. Para isto basta selecionar a opção "capturar saída no painel saída". Depois que a compilação termina, o Notes lê a saída e vê se o compilador informou algum erro. Você poderá navegar pelo erros então indo em  "Localizar >> próximo problema" e "Localizar >> problema anterior". O Notes também vai mostrar um ícone de erro na gutter ao lado das linhas onde o compilador tiver achado um erro. Você também pode navegar pelos erros clicando duas vezes na linha do erro no painel saída. O Notes é capaz de interpretar os erros de compilação da maioria dos compiladores automaticamente. No caso do Notes não conseguir entender os erros do seu compilador, veja no próximo capítulo como configurá-lo para que ele passe a entendê-los.


Rodando o aplicativo compilado

Após você compilar o código, vai querer rodar o seu programa. Para rodar executáveis do windows, você deve criar outro item no menu executar e então preencher no campo "executar" o seguinte: 
%path%%FileTitle%.exe
Para rodar um arquivo "class", ou seja, Java bytecode você deve colocar o "java.exe" (exemplo: "C:\Arquivos de programas\javasdk142\bin\java.exe") no campo executar e %FileTitle% no campo "argumentos".
