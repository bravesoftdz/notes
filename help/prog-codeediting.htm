Editando código


Autocompletando

O Notes pode completar tags e símbolos como ', ", {, [ e ( automaticamente. Isto diminuí a quantidade de caracteres que você precisa digitar e evita que você esqueça de fechar alguma tag, uma string, comentário, etc. ajudando assim a evitar que ocorram bugs nos códigos que você escreve. Você pode habilitar e desabilitar estas features no diálogo de preferências.

Visualizando o código

O Notes ajuda e muito a visualizar o seu código. O editor possuí uma coloração de sintaxe poderosa baseada em XML que pode ser completamente customizada. Além disto o Notes pode marcar a linha atual com uma cor de fundo diferente (você pode desabilitar isto no diálogo de preferências setando a cor da linha ativa para a mesma cor do fundo do editor) e mostrar símbolos coincidentes como ( e ), { e } e [ e ]. Outra feature útil é o marcador de escopo que mostra na gutter o bloco de código em que você está, ajudando a ler o código e a identificar blocos de código mal fechados.

Outros comandos

Outros comandos úteis quando se trabalha com código são Editar >> Comentar e Editar >> Descomentar que comentam e descomentam trechos de código selecionado ou a linha atual.
