Criando Websites


O Notes traz grandes facilidades para quem quer criar websites em HTML, ASP .Net, PHP, Perl, etc. A primeira coisa que você vai querer fazer é configurar o Notes para fazer preview dos sites que você desenvolve.

O Notes possuí um editor cheio de features para ajudá-lo a criar websites. O Notes pode fechar tags automaticamente, identar o código, colorir a sintaxe, etc. Veja mais sobre sobre edição de arquivos aqui. Você deve olhar também o menu "Comandos". Ele possuí comandos para veririficar HTML e coverter HTML para XHTML usando o HTMLTidy, converter carateres especiais (como letras com acentuação) para os códigos especiais do HTML, etc. Outra coisa que provavelmente você achará muito útil é o suporte a quebra automática de linhas (WordWrap) que pode ser ativado no menu "opções".

Se você programa para web, você sabe (ou ao menos deveria saber) programar em JavaScript. A grande notícia é que o Notes pode executar macros escritas em JavaScript, isto é, você pode criar novos comandos e adicionar novas funcionalidades ao Notes usando uma linguagem que você já conhece.

Outra coisa que você vai gostar muito é a biblioteca do Notes. O Notes possuí uma bblioteca de códigos para cada uma das linguagems de programação. A biblioteca de HTML é uma das mais completas. Você pode acessar a biblioteca através do painel biblioteca. No painel você pode usar o menu de contexto dele (clicando com o botão direito do mouse) para criar novos items, deletar items, etc. Além da biblioteca, o Notes também possuí suporte a Templates. De fato, o Notes sempre cria novos arquivos através de templates, como você já deve ter percebido ao usar o diálogo para novos arquivos. Você pode criar os seus próprios templates para os sites que você está desenvolvendo.
