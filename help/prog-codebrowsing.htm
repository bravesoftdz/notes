Navegando no Código


Marcadores

Quantas vezes você já precisou ir para um outro trecho de código e depois voltar para onde você estava? E quantas vezes você perdeu tempo procurando o tal trecho onde você estava?! Bom, os marcadores servem para evitar que isto volte a acontecer. Você pode marcar uma linha usando Localizar >> adicionar/remover marcador e depois voltar para a linha marcada usando a opção "ir para marcador" ou os comandos "marcador anterior" e "próximo marcador". Usando marcadores você não se perde mais no seu próprio código. O Notes pode lembrar dos marcadores que você colocou em um arquivo entre uma sessão e outra, para isto mantenha a opção "lembrar da última posição de edição e dos marcadores" habilitada no diálogo de preferências.

Ir para linha, ir para coluna, Avançar e Voltar

O Notes pode também pular para uma linha ou coluna do texto. Para isto usa a opção "ir para a linha" ou "ir para a coluna" do menu Localizar. Basta entrar o número da linha/coluna no diálogo que irá aparecer para ir para onde você quer.

As opções Voltar e Avança do menu localizar poderão também ajudá-lo a navegar pelo código. Elas levam você para lugares que você já visitou no documento, podendo até mesmo reabrir um documento que você já fechou para levá-lo aonde você quer ir.

Achando o que você procura

O Notes possuí suporte a busca e substituição avançada no texto. Estas ferramentas podem ser acessadas através dos comandos Localizar >> Localizar e Localizar >> Substituir. Após fazer uma busca você pode usar também os comandos "localizar próxima" e "localizar anteiror" para buscar as próximas ocorrências com o diálogo de busca fechado. Além de buscar no documento atual, o Notes pode também buscar dentro de diretórios ou em todos os arquivos abertos.
