Introdução


Macros são úteis sempre quando você precisa repetir um certo número de tarefas no Notes. Você pode escrever macros poderosas em JavaScript ou gravar macros usando os menus do Notes.


Gravando e executando macros

A gravação e execução de macros do Notes funcionam da mesma maneira que em qualquer outro editor:

1) Você aciona a gravação da macro indo em Macros >> Gravar macro/Para gravação. A Statusbar mostrará a mensagem "gravando" para que você possa saber que a macro está sendo gravada;
2) Você digita uma série de comandos (insere letras, deleta texto, cola texto da área de transferência) que você quer que sejam repetidos pelo Notes. Como nos outros editores, ações feitas pelo mouse no editor não são gravadas, então você deve selecionar texto sempre usando o teclado;
3) Você pára a gravação indo novamente em Macros >> Gravar macro/Para gravação. A Statusbar não mostrará mais a mensagem "gravando";
4) Você vai em Macros >> Executar macro para que o Notes repita as ações que você gravou.

Após gravar uma macro você pode salvá-la indo em Macros >> Salvar macro como.

Dica: para gravar uma macro que faça uma operação em cada uma das linhas do arquivo, faça a macro ir para a próxima linha do arquivo depois que ela terminar de modificar a linha em que ela está. Assim você pode manter a tecla de atalho que você estiver usando para o comando "executar macro" pressionada para que as macros modifiquem todas as linhas do arquivo.

Outra dica: as macros gravadas são traduzidas para JavaScript. Você pode editar uma macro depois de salvá-la dentro do Notes, já que na verdade ela é um arquivo com código dentro :)

Aviso: as macros podem chamar diálogos do Notes mas não podem entrar textos nem manipular os controles, já que a API para JavaScript do Notes ainda não fornece meios de fazer isto.
